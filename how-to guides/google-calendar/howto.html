How to Interact with Google’s Cloud-Based Calendar with Python

by Manhing Lei

Introduction and Set Up

Introduction

Google Calendar is a popular cloud-based application for everyday people and for use as a corporate calendar. If you have a Google account, you have a Google Calendar.

This tutorial will teach you how to create a simple program to interact with one or more calendars at once. More specifically, we will concentrate on the events of public Google calendars. An event is anything on a calendar, like a lunch meeting on a day and time.

We will use HTTP requests to use the API and write the program with Python. This particular implementation bypasses the use of authorization and authentication. As mentioned earlier, the limitation to this is that the calendar we are accessing must be public.

If you want to interact with (limited access) calendars, use the OAuth2.0 way. This video will help you get started, but this tutorial will not cover that particular implementation.

If this tutorial is not for you, please feel free to click here for a general overview and you can go from there.

As you can see, Google provides documentation to various other implementations, so you can pick from a variety of languages in which to develop your program. If you choose to continue with this particular implementation, I encourage you to visit this page and skim through.

Once completed, this program can be passed on to your friends or colleagues without downloading anything else. Other implementations will require you to download a client for OAuth2.0 authentication.

Set Up

The first step is to sign up for an API key. This key will be used instead of OAuth2.0 authentication. Go here and follow the prompts. If you mess up, don't worry. You can repeat the process. The goal is to end up with a "Server" API key. Once done, if you look up your list of credentials, you should have one of these:


In regards to usage limits, quotas and expiration of the API key, I could not find any documentation. I assume there has to be some limit. With the amount of testing I've done in a day, I have easily used the key 20 times in one day. This is plenty for any normal user.


Now, let's make our Google calendar public. Go to your Google calendar app. Find the "My calendars" drop down and select "Settings".


Click on "Share this calendar".



Check the checkbox next to "Make this calendar public".



Assuming you have basic Python experience, we are all set. Let's get started!
