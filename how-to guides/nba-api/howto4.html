How to use NBA.com's API

by Manhing Lei

Explore to Learn More

At this point, I can spoon feed you by listing the formats for each and every parameter value, such as any time a season is used as a parameter, you have to format the value as "2014-15", not "2014-2015", not "14-15", not "2014-5". But I won't do that. I don't want to give you the fish. I want to teach you how to fish. To do this, let's explore the website itself.

I am still at Quincy Acy's player page. If you scroll down, you will see plenty of statistics. Where did they come from? Did a web developer type all those numbers in? Of course not, don't be ridiculous! These statistics are dynamically populated by having a server do exactly what we did manually in the previous page.

In your browser, right-click anywhere on the page. In Chrome, select Inspect. In FireFox and Internet Explorer, select Inspect Element. Go to the Network tab. And if you have it, select XHR. This is where you can watch the magic happen.

While we're still on Quincy's stats page, reload it. Oooooh, do you see all that? This is how it looks in FireFox. Chrome should look similar.

These are the things that loaded to make this page what it is. The second, fourth, and sixth items were queries to load the statistics. Does the format look familiar? It should. They have the endpoint, followed by a slash and a question mark. Then comes the parameters and then the values. Each parameter name and value pair are separated by an ampersand. So now, you know the format of IsOnlyCurrentSeason is an integer or a boolean... likely a boolean. LeagueID is a two-digit value, 00 being a valid value. I am pretty sure 00 represents the Men's NBA. There is likely a value for Women's NBA (WNBA) and the Developmental League (NBA D-League), maybe others.

So what am I trying to say? Elias Uriegas and Nick Bottomley, the people whose work I linked earlier, did not work for NBA.com (or at least not to my knowledge). They merely clicked around the site watching what queries were sent each time a page loaded and gathered that information. You can too. Now you know how to fish!
